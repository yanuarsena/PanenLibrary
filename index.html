Setup Koneksi PWA

Masukkan URL Web App Google Apps Script asli Anda yang berakhiran /exec untuk mengaktifkan sinkronisasi PWA.

*Pilihan ini akan disimpan langsung di memori HP Anda. Anda dapat mengubahnya kapan saja melalui menu Profil.

Jika layar tetap tertahan memuat, browser Anda mungkin memblokir sistem otorisasi login Google di dalam aplikasi.

Koneksi Terputus

PANEN Mobile memerlukan koneksi internet aktif untuk menarik data keuangan Anda secara real-time dari server Google.
